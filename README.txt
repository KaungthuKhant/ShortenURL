URL Shortener Application
=========================

This is the latest stable version of the URL Shortener application. Last tested and verified on Aug 28.

Getting Started
---------------
To run this program:
1. Open a new terminal
2. Enter the command: npm run devStart
3. Access the website at http://localhost:8800/

Features
--------
1. User Authentication:
   - User registration
   - User login
   - Email confirmation for new accounts

2. URL Shortening:
   - Generate short URLs for long links
   - Option to create custom short URLs
   - QR code generation for shortened URLs

3. URL Management:
   - Click tracking for each shortened URL
   - Set expiration dates for URLs
   - Email notifications for click count milestones
   - Email reminders for URLs about to expire

4. Security:
   - Password strength validation
   - Secure session management

5. User Interface:
   - Home page with URL checker for non-logged-in users
   - Dashboard for registered users to manage their shortened URLs

6. Database:
   - MongoDB integration for data persistence


Future Improvements
-------------------
- Implement session timeout to enhance security
- Add more customization options for shortened URLs
- Improve analytics for URL usage
